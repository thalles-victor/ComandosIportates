# Intalação do Styled Components com TypeScirpt

  Para React: 
  ```console
      yarn add styled-components
   ```
  
  ```console
      yarn add -D @types/styled-components 
  ```
  
  Para o React Native
  
  ```console
      yarn add styled-components
  ```
  
  ```console
    yarn add -D @types/styled-components-react-native
  ```
