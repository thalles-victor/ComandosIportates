# Configuração do Styled Components com TypeScirpt para React e React Native

  Para React: 
  ```console
      yarn add styled-components
   ```
  
  ```console
      yarn add -D @types/styled-components 
  ```
  
  Para o React Native
  
  ```console
  yarn add styled-components
  ```
  
  ```console
yarn add -D @types/styled-components-react-native
  ```
