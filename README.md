# Intalação do Styled Components com TypeScirpt

  Para React: 
  ```console
      yarn add styled-components </li>
   ```
  
  ```console
      yarn add @types/styled-components -D </li>
  ```
  
  Para o React Native
  
  ```console
      yarn add styled-components
  ```
  
  ```console
    yarn add @types/styled-components-react-native
  ```
